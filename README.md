# react-starter

React bootstrap includes Webpack, React, Jest and Redux

## Install

```
$ git clone https://github.com/felipemrodrigues/react-bootstrap
$ npm install
```

## Commands

#### build

```
$ npm run build
```

#### dev

```
$ npm run dev
```

Runs application in the browser (http://localhost:8080).

#### test

```
$ npm run test
```

## Features

* ES2015 & ES2016 support
